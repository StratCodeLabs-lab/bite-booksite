# GitHub'a 5 Satırda Yükle

## 1. GitHub'da Repo Oluştur
https://github.com/new → `bite-booksite` → Create

## 2. Terminal'de Şu Komutları Çalıştır

```bash
git init
git add .
git commit -m "Bite BookSite v1.0"
git branch -M main
git remote add origin https://github.com/KULLANICI_ADINIZ/bite-booksite.git
git push -u origin main
```

## 3. GitHub'da Settings → Pages → GitHub Actions Seç

Bitti! Siteniz şurada:
```
https://KULLANICI_ADINIZ.github.io/bite-booksite
```
